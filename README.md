# kata-sant-jordi-2026
**Kata 76 per a especialitats web 23-4-26**
És Sant Jordi!

Roses i llibres per tothom!

Crea un programa que et permeti obtenir el teu regal random:

- Has de triar si vols un llibre o una rosa
- Hi ha 5 llibres possibles (cadascun amb el seu títol)
- Hi ha 5 roses possibles (cadascuna amb el seu color)
